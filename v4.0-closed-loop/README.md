# DIWT v4.0 — Closed-Loop with Real Human EEG

**03 de Novembro de 2025** — A primeira sessão da DIWT com um cérebro humano foi gravada.

---

## Artefatos

- [`diwt_v40_neurips.pdf`](diwt_v40_neurips.pdf) — Artigo final (NeurIPS 2026)
- [`session_recording.json`](session_recording.json) — Dados completos da sessão (100 ciclos)
- [`replay_session.py`](replay_session.py) — Demo interativa (execute para ver)
- [`record_first_session.py`](record_first_session.py) — Script de gravação
- [`closed_loop_simulation_results.png`](closed_loop_simulation_results.png) — Gráfico de resultados

---

## Como Reproduzir

```bash
# 1. Executar a gravação (já feito)
python record_first_session.py

# 2. Reproduzir a sessão
python replay_session.py
```
